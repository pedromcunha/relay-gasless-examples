# Relay Gasless Examples

Example implementations for gasless swaps and bridges using [Relay](https://relay.link).

For the full guide, see the [Gasless Swaps documentation](http://docs.relay.link/features/gasless-swaps).

## Examples

| Example | Description |
| --- | --- |
| [4337-gasless](./4337-gasless) | Gasless cross-chain bridging using an ERC-4337 smart account with Relay `/execute` |
| [7702-batch-executor](./7702-batch-executor) | Gasless swaps using EIP-7702 delegation to Calibur (Uniswap's batch executor) |
| [byo-eoa-permit](./byo-eoa-permit) | Permit-based cross-chain bridging with a user's existing EOA wallet |
